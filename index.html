Hexo instructions

Follow me

  1. 安装Node
    Node

  2. 安装Git
    Git

  3. 注册Github账号

    *建立与你用户名完全对应的Repository(your_user_name.github.io)

    *添加SSH公钥到(Account settings -> SSH Keys -> Add SSH Key)

    如何添加：在Git安装所在文件夹名称右击打开Git bash
设置：git config --global user.email "840984768@qq.com"
      git config --global user.name "mademoiselle-susane"
生成密钥：
      ssh-keygen -t rsa -C "840984768@qq.com"
      默认enter，在 C:\Users\自己电脑用户名\.ssh 目录下将生成的密钥拷贝到Git安装目录下
      用文本编辑器打开ssh.pub文件，拷贝其中的内容，将其添加到Github账户中的Add SSH Key下的Key中，title随意。
验证：ssh -T git@github.com

  4. 安装hexo

    • npm install -g hexo

      安装完成后，在你喜爱的文件夹下（如H:\hexo），执行以下指令(在H:\hexo内点击鼠标右键，选择Git bash)
      Hexo 即会自动在目标文件夹建立网站所需要的所有文件。

    • hexo init

      根据提示可能会需要执行下面命令（安装依赖包）
      npm install

      本地查看：
      现在我们已经搭建起本地的hexo博客了，执行以下命令(在H:\hexo)，然后到浏览器输入localhost:4000看看。
      hexo generate
      hexo server

  5. 部署

    文本编辑打开 _config.yml(在H:\hexo下)，在部署时，要把下面的 mademoiselle-susane 都换成你的账号名。
    deploy:

    type: git
repository: https://github.com/mademoiselle-susane/mademoiselle-susane.github.io.git
branch: master

    执行：

    hexo generate
hexo deploy

此时可能需要输入username：你的github账号名，非邮箱
密码：github账户密码

    可能报错：deploy not found：github
    这时运行下npm install hexo-deployer-git —save
    再执行

    hexo generate
hexo deploy

Attention：
记住：每次修改本地文件后，需要hexo generate才能保存。每次使用命令时，都要在H:\hexo目录下。
Okay,我们的博客已经完全搭建起来了，在浏览器访问mademoiselle-susane.github.io就能看到你的成就了！
